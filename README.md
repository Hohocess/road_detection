# road_detection / LaneDetectionSystem

Road / lane detection system project.

Contents:
- Python scripts for preprocessing and running detection (`main.py`, `yolo_process.py`, `cv2_process.py`, etc.)
- Dataset folders (`dataset/`, `CULane/`) are excluded from the repo by default via `.gitignore` to avoid large files.

To upload full datasets or model weights, add them separately (LFS or releases) or remove corresponding lines from `.gitignore`.
